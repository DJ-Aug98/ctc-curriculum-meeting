### Stage 1 — Desired Results

**Competency Cluster**
Duty Area B — Workplace Readiness / Tasks B1–B6
*Drawn from program DACUM task inventory: follow safety protocols, use personal protective equipment, communicate with supervisors and peers, read and interpret workplace documentation, apply industry-standard quality checks, maintain a professional work environment.*

**Industry Credential Alignment**
[Relevant credential or certification framework], Domain 1: Professional and Technical Foundations

**Enduring Understanding**
Safe, professional practice is not a separate layer added onto technical work — it is built into how skilled practitioners approach every task, every time.

**Essential Questions**
- What does it actually mean to work like a professional — and how would you know if you were?
- Why do industry standards exist, and what happens when they aren't followed?
- How do you communicate a problem you don't fully understand yet to someone who can help you solve it?

**Knowledge and Skills**

*Students will know:*
- Program-specific safety requirements and relevant regulatory standards (e.g., OSHA, program-area equivalents)
- The purpose and proper use of personal protective equipment for this program area
- How to read and interpret standard workplace documentation (work orders, task sheets, inspection checklists)
- The professional expectations of entry-level practitioners in this field

*Students will be able to:*
- Demonstrate correct safety protocols before, during, and after a task
- Select and properly use appropriate personal protective equipment for a given task
- Identify and report a safety hazard using program-standard documentation
- Complete a task sheet or work order accurately and legibly
- Communicate a work update or concern clearly to a supervisor or peer

### Stage 2 — Assessment Evidence

**Performance Task**
Students will complete a simulated workplace scenario: given a job setup with 3–4 intentional safety and documentation gaps, students will identify each issue, correct what can be corrected, document what needs to be escalated, and complete the associated work order. Performance is evaluated against the program's professional practice checklist.

**Assessment Criteria / Rubric Reference**
Program Professional Practice Checklist (see attached) — evaluates safety compliance, PPE use, documentation accuracy, and communication under observation.

**Other Evidence**
- Safety protocol demonstration (observation, first week)
- Daily task sheet completion (formative, ongoing)
- Peer walk-through debrief (mid-unit)
- Workplace documentation quiz (written, lesson 6)

### Stage 3 — Learning Plan

**Sequence Overview**
The unit moves from awareness → skill practice → independent application. Early lessons establish the "why" behind professional standards; mid-unit lessons build procedural fluency; final lessons require students to apply and self-monitor without prompting.

| Lesson | Title | Focus | Type |
|---|---|---|---|
| 1 | What Does Professional Practice Look Like? | Anchor the enduring understanding; review exemplar and non-exemplar work | Introduction |
| 2 | Safety Standards in This Program Area | Program-specific safety requirements, OSHA overview, regulatory purpose | Skill-building |
| 3 | PPE — Selection, Use, and Limits | Hands-on PPE identification and proper use | Skill-building |
| 4 | Reading Workplace Documentation | Work orders, task sheets, inspection checklists — read and interpret | Skill-building |
| 5 | Completing Workplace Documentation | Fill out work orders accurately; common errors and how to catch them | Skill-building |
| 6 | Documentation Quiz + Debrief | Written assessment; class debrief on common mistakes | Assessment |
| 7 | Communicating at Work | Verbal and written communication norms; reporting hazards and updates | Skill-building |
| 8 | Peer Walk-Through | Students evaluate a peer-prepared workspace against the checklist | Application |
| 9 | Performance Task Prep | Review scenario format; self-assessment against criteria | Application |
| 10 | Performance Task | Simulated workplace scenario; evaluated against professional practice checklist | Assessment |

**Key Resources and Materials**
- Program safety standards documentation (current edition)
- PPE kit for program area
- Sample work orders and task sheets (blank and completed examples)
- Professional practice checklist
- Scenario setup materials (lesson 10)

**Safety and Compliance Notes**
All hands-on lessons require PPE per program protocols. Lesson 3 includes equipment demonstration — instructor must confirm student understanding before any independent handling. Report any identified safety hazards to [campus safety coordinator] per standard protocol.

**Differentiation Notes**
- Written documentation tasks available in simplified format for students with IEP/504 accommodations
- Peer walk-through (lesson 8) can be structured as a supported pair activity for students who need additional scaffolding
- Extension: students who complete the performance task early can document a second scenario from a supervisor's perspective
