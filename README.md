# CTC Curriculum Infrastructure — Director Meeting
## CCIU Internal Project

### Project Overview
A 60-minute facilitated meeting with the CTC Director to present a curriculum infrastructure framework — addressing findings from a winter SREB audit and teacher focus groups. The goal is to align on a direction for curriculum development support across all three CTC campuses.

### Meeting Non-Negotiables
1. Curriculum infrastructure — a structured, consistent home for curriculum across all three campuses
2. Curriculum writing protocol — a process for how curriculum gets built, including the teacher learning required to use it

### Key Context
- **Audit:** SREB instructional audit completed winter — findings include inconsistency across campuses, curriculum not formally documented, misalignment to competencies
- **Focus groups:** Newer teachers lack a shared mental model of unit vs. lesson — working at lesson level without a larger instructional arc
- **Parallel work:** Active curriculum development project with a traditional school district — proof of process
- **Director's signal:** Loosely mentioned building trades as a possible starting point

### Facilitation Team Notes
- Two facilitators recommended — one driving conversation, one navigating dashboard
- Dashboard should be navigable non-linearly to respond to director's reactions
- Building Trades example should be ready but not led with
- SREB audit artifacts (curriculum maps, lesson plans) needed as physical or displayed props
